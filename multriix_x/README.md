## 🧠 PROJECT OVERVIEW
MULTRIIX X is a unified AI control platform that lets the user:
- Run, mix, and switch between Qwen, Mistral, Ollama models, and GLM locally
- See a LIVE BRAIN — a real-time neural network visualization showing token generation, attention layers, and thought flow as it happens
- Edit the AI's system prompt, memory, personality, and parameters LIVE without restarting
- Have FULL PC CONTROL — run shell commands, manage files, control processes, open apps, all from the dashboard
- Configure everything from raw model weights to output formatting
- Mix models mid-conversation (e.g. use Qwen for reasoning, Mistral for writing)
- Monitor everything: CPU, GPU, RAM, VRAM, temperature, speed
