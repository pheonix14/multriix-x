Creator & Info

Vision, privacy, security, and developer notes for MULTRIIX X

v1.0.0

[★] Creator's Note & Vision

Welcome to MULTRIIX X! Built and guided by ZeroX (aka pheonix14 on GitHub). This project is built because I believe in a future where users fully own, modify, and control their AI models, instead of being monitored, restricted, or charged high fees by mega-corporations.

Local AI represents true user sovereignty, absolute privacy, and total flexibility. Many people do not have the high-end resources to train their own models or build custom systems, which is why we created MULTRIIX X. Our dream is to collaborate as an open-source community to build the world's best locally-running AI ecosystem that can serve as a base for developers and creators everywhere.

"Not everyone has the money to pay for subscriptions. I am also broke, so let's build the ultimate local AI system together!" — ZeroX

[★] Follow on GitHub [❤] Donate & Support

[🛡] Privacy Policy

MULTRIIX X guarantees absolute data sovereignty:

  • 100% Offline & Local: Every chat, parameter change, file edit, and shell command runs entirely on your local machine.
  • Zero Telemetry: There are no analytics, trackers, or hidden reporting channels.
  • Secure Sandbox: You control which folders and files are accessible. No third-party servers see your prompts.

[🔐] Hacker-Proof Security

Hardened local system isolation:

  • Localhost Restriction: The server strictly binds to 127.0.0.1 (localhost) rather than 0.0.0.0, blocking any network-level interception.
  • Resilient Port Manager: Scans port bounds dynamically on boot to isolate running nodes from port-hijacking or conflicts.
  • Automatic Fallbacks: Seamlessly shifts to standalone GLM/HuggingFace or Ollama standby if primary local nodes are offline.
