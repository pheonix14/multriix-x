# NeuralDesk — by Pheonix14 / ZeroX

**Your personal, 100% offline AI model manager and chat interface.**

## Quick Start

```bash
cd multriix_x/neuraldesk
python app.py
```

That's it. Your browser opens automatically.

## What It Does

- 💬 **Chat** — Stream conversations with any local Ollama model
- 🤖 **Models** — Browse, download, delete AI models with one click
- ⚙️ **Settings** — Visual sliders and toggles (no code needed)
- 📁 **Files** — Browse and edit any file including Modelfiles
- 📊 **Monitor** — Live CPU, RAM, GPU, disk + process table

## Requirements

- Python 3.10+
- [Ollama](https://ollama.com) installed and running
